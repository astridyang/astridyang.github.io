#### 2.10 CSS中的值和单位
1. em 元素字体⼤⼩

    ``` height: 2em; ```  
    height = 20px*2 = 40px;   
2. rem html字体⼤⼩  
    ```height: 2rem; ```  
    height = 16px*2 = 32px;(16px为html标签的font-size, 默认值)
